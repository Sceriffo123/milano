Cronologia verticale con linea rossa e nodi: racconta la continuità del metodo nel tempo.

```jsx
<Timeline items={[
  { year: "2017", children: "Deriso perché dicevo che l'abusivismo si combatte con gli atti." },
  { year: "2022", children: "24 maggio: l'accordo nazionale con la multinazionale." },
  { year: "2026", children: "Le condizioni vengono dettate dentro casa nostra." },
]} />
```

Le date restano in rosso; il testo asciutto, una riga per voce.
