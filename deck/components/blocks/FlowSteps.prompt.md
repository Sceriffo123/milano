"La macchina": la sequenza di atti protocollati. Ogni passo un blocco; l'ultimo, pieno di rosso, è l'esito.

```jsx
<FlowSteps steps={[
  { n: 1, title: "SEGNALAZIONE", note: "dal territorio" },
  { n: 2, title: "ISTANZA", note: "verifica e controllo" },
  { n: 3, title: "SOLLECITO", note: "con termini" },
  { n: 4, title: "DIFFIDA", note: "ad adempiere" },
  { n: 5, title: "ESPOSTO", note: "autorità superiori" },
]} />
```

Titoli in maiuscolo, note minime. Rappresenta una procedura, non una protesta.
